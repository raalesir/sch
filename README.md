# README

The exercise is to write a command-line text search engine.

## Usage:

1. create virtual env for python3
```bash
python3 -m venv sch
source sch/bin/activate
```
1. install `pytest`
```bash
pip3 install -r requirements.txt
```
1. run tests
```bash
pytest search.py
```
Both should pass
1. launch text search engine:
```bash
./search.py
```
